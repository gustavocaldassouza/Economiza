# Economiza
> **See where your money goes. Securely.**

## About
Economiza is a local-first, privacy-centric personal finance application designed to demystify daily spending without compromising user data. By focusing on rapid data entry via on-device intelligence and clear, instant visualization, Economiza helps users identify spending leaks and build better financial habits. Unlike traditional finance apps, Economiza operates entirely offline, ensuring that sensitive financial details never leave the user's device.

## Core Features
*   **Smart Transaction Entry**: Rapidly log income and expenses with amount, date, and notes, using OCR to automatically scan and extract transaction details.
*   **Custom Categorization**: Organize spending with fully editable categories, icons, and color-coded tags.
*   **Interactive Dashboard**: Visualize financial health through dynamic pie charts and bar graphs that break down spending by category.
*   **Budget Management**: Set monthly spending limits for specific categories (e.g., "Groceries") with visual progress bars.
*   **Recurring Payments**: Automate the tracking of fixed monthly expenses like rent, subscriptions, and utilities.
*   **Data Export**: Functionality to download transaction history as CSV or PDF files for external analysis or backup.

## The "Privacy-First" Edge
*   **Local-First Privacy Architecture**: Complete data sovereignty where all financial records are encrypted and stored solely on the user's device, requiring no internet connection or cloud account.
*   **On-Device Edge OCR**: Uses local machine learning to scan and extract receipt data instantly without ever uploading images to a third-party server, ensuring faster processing and total privacy.
*   **Predictive Spending Forecasts**: Analyzes past spending habits to project future end-of-month balances, warning users in real-time if a current purchase will cause them to miss a savings goal.

## Tech Stack
*   **Language**: Java
*   **Platform**: Android
*   **Build System**: Gradle

## Privacy Policy
Economiza is designed with privacy as its core principle.
*   **No Data Collection**: We do not collect, transmit, or store any of your personal financial data on our servers.
*   **Local Storage**: All data is stored locally on your device.
*   **Offline Functionality**: The app functions entirely offline.
